üìä What Happens During Deployment

  1. Metadata Upload: Token information and image uploaded to IPFS
  2. Mint Account: Creates new token mint account on Solana
  3. Initialize Mint: Sets up token with decimals and authority
  4. Token Account: Creates associated token account for creator
  5. Mint Supply: Mints the total supply to creator's account
  6. Add Liquidity: Creates trading pool with initial SOL liquidity
  7. Platform Fee: Pays deployment fee to platform wallet
  8. Finalize: Removes mint authority and completes setup
